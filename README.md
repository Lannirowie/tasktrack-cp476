# tasktrack-cp476 - CP476 Milestone 2

## Project Overview
TaskTrack is a simple task management web application that allows users to do the following:
1. Register and login
2. View tasks
3. Add new tasks
4. Mark tasks as complete
5. Delete tasks


The Frontend workflow is fully functional using mock data

## How to Run the Frontend
1. Download the repository
2. Open the project folder in VS Code
3. Open 'frontend/login.html' with live server
4. Use the application through the browser


## Frontend Features Implemented
1. Register page with password validation
2. Login page that redirects to the tasks page
3. Tasks list page
4. Add Task form
5. Complete task toggle
6. Delete tasks
7. Data persistence using localStorage


## Backend
The backend folder includes the structure for the following:
1. Runnable server entry point
2. Routes
3. Controllers
